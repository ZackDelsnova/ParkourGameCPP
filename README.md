# ParkourGame

# Requirments

- SDL3
- GLAD
- GLM

Download the dependencies and place them in the `include` folder.

Place glad.c file in the `src` folder.

Place SDL3.lib in the `lib` folder.

Place SDL3.dll in the executable folder.
